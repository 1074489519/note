- JavaScript 中的数字都使用 IEEE-754 标准以 64 位格式存储。但是在位操作中，数字被转换为有符号的 32 位格式。即使需要转换，位操作也比其他数学运算和布尔操作快得多。
- # 取模
	- 由于偶数的最低位为 0，奇数为 1，所以取模运算可以用位操作来代替。
	- ```js
	  if (value % 2) {
	  	// 奇数
	  } else {
	  	// 偶数 
	  }
	  // 位操作
	  if (value & 1) {
	  	// 奇数
	  } else {
	  	// 偶数
	  }
	  ```
- # 取整
	- ```js
	  ~~10.12 // 10
	  ~~10 // 10
	  ~~'1.5' // 1
	  ~~undefined // 0
	  ~~null // 0
	  
	  ```
- # 位掩码
	- ```js
	  const a = 1
	  const b = 2
	  const c = 4
	  const options = a | b | c
	  ```
	- 通过定义这些选项，可以用按位与操作来判断 a/b/c 是否在 options 中。
	- ```js
	  // 选项 b 是否在选项中
	  if (b & options) {
	  	...
	  }
	  ```
