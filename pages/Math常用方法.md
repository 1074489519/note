- | 方法 | 作用 |
  | ---- | ---- | ---- |
  | Math.max(...arr) | 取arr中的最大值 |
  | Math.min(...arr) | 取arr中的最小值 |
  | Math.ceil(小数) | 小数向上取整 |
  | Math.floor(小数) | 小数向下取整 |
  | Math.round(小数) | 小数四舍五入 |
  | Math.sqrt(num) | 对num进行开方 |
  | Math.pow(num, m) | 对num取m次幂 |
  | Math.random() * num | 取0-num的随机数 |
